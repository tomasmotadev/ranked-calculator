# Ranked Matches Calculator

This project calculates the balance of ranked matches for a player based on wins and losses, and determines the player’s rank according to their win count.

## How to use

1. Clone the repository.
2. Run the main script: ```bash python main.py```

3. Enter the number of wins and losses when prompted.

## Project structure
```calculator.py``` — core logic of the calculator <br>
```main.py``` — user interaction script <br>
```tests/``` — unit tests for validating the functions

## Requirements
    Python 3.x

## Running tests
To run tests, execute:
  ```bash python -m unittest discover tests```
